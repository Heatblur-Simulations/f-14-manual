## Contenu

Cette documentation contient le manuel de l'avion F-14 Tomcat. Le contenu
est regroupé par variante d'avion :

- Le chapitre [F-14A/B](f14ab.md) est le manuel principal, répertoriant et expliquant tous les
  systèmes. Une attention particulière est accordée aux variantes
  - F-14A-95-GR (Exportation),
  - F-14A-135-GR (début),
  - F-14A-135-GR (tardif) et
  - F-14B.
- Le chapitre [F-14B(U)](f14bu.md) détaille ensuite tous les changements introduits
  avec la mise à niveau F-14B en particulier.
