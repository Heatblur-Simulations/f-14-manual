# Cadre de pare-brise droit

## ECM Témoins d'avertissement

![rwr attention](../../../img/cockpit_detail_pilot_rwrcaution.jpg)

Feux d'avertissement connectés à l'ALR-67 indiquant différents types de menaces.

### ALR-67

| Indicateur | Fonction |
| --- | --- |
| SAM | Éclairage constant lors de la détection du verrouillage à partir d'un radar de suivi SAM. Clignote lorsqu'un lancement de missile est détecté. |
| AAA | Éclairage constant lors de la détection du verrouillage à partir d'un radar de suivi AAA. Clignote lorsque le tir AAA est détecté. |
| AI | Éclairage constant lors de la détection du verrouillage d'un radar intercepteur aéroporté. |

### ALR-45

| Indicateur | Fonction |
| --- | --- |
| SA TRK | S'allume fixement chaque fois qu'un radar de suivi SAM est reçu. |
| SAM | Indication d'avertissement Master SAM, lumières fixes pour MA (alerte missile), clignote pour ML (lancement de missile) |
| AI/AAA | S'allume en continu chaque fois qu'un radar AI/AAA ambigu est reçu. |
| AI | Les lumières restent fixes chaque fois qu'un radar intercepteur aéroporté est reçu. |
