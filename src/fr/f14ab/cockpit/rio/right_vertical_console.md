# Console verticale droite

## Affichage des contre-mesures électroniques (ECMD)

![mdi](../../../img/cockpit_detail_rio_mdi.jpg)

Affichage utilisé pour les informations de navigation et AN/ALR-45 ECM (uniquement sur F-14A avec
AN/ALR-45).

Possède un bouton de contrôle de la luminosité, un bouton de test et un indicateur BIT indiquant
l'état de l'écran (noir fixe lorsqu'il est opérationnel, affichant des drapeaux blancs lorsqu'il
indique une condition d'échec).
