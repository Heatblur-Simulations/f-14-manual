# Tableau de bord droit

## Horloge

![horloge](../../../img/cockpit_detail_rio_clock.jpg)

Horloge à remontage mécanique.

Le bouton vent/réglage (<num>1</num>) est situé dans le coin inférieur gauche.

- Tournez dans le sens des aiguilles d’une montre pour remonter l’horloge.
- Tirez et faites pivoter pour régler les aiguilles des heures et des minutes.

Le contrôle du temps écoulé (<num>2</num>) est situé dans le coin supérieur droit et
est utilisé pour démarrer, arrêter et réinitialiser le compteur de temps écoulé d’une heure.

## Indicateur ALR-67

![alr-67](../../../img/cockpit_detail_rio_alr-67.jpg)

Affiche les émetteurs radar détectés par le récepteur d'avertissement radar ALR-67.

### Bandes d'affichage des menaces

- Bande non létale (<num>2</num>) – Affiche les émetteurs qui ne sont pas considérés comme un danger
  immédiat.
menace due à la portée ou au manque de capacité des armes.
- Bande mortelle (<num>3</num>) - Affiche les émetteurs capables d'engager leurs propres
avion, mais ce n’est pas le cas actuellement.
- Bande critique (<num>4</num>) - Affiche les menaces directes contre votre propre avion.
Systèmes capables d'engager son propre avion et de montrer l'intention actuelle de le faire
donc.

### Cercle d'état du système

Le cercle d'état du système (<num>1</num>) est divisé en trois zones.

**La zone I (quadrant supérieur gauche)** affiche les symboles de priorisation des menaces :

- N-Normal.
- I - IA, intercepteurs aéroportés prioritaires.
- A - AAA, artillerie anti-aérienne prioritaire.
- U - Émetteurs inconnus prioritaires.
- F - Les émetteurs amis affichés en plus des menaces.

**La zone II (quadrant supérieur droit)** indique l'état du mode limité.

- (Vide) - Mode limité non sélectionné.
- L - Mode limité sélectionné. Seules les six menaces les plus prioritaires sont affichées.

**La zone III (moitié inférieure)** affiche l'état du système et les informations de décalage :

- (Vide) - Fonctionnement normal.
- B - Panne BIT.
- T - Surcharge thermique.
- O - Affichage du décalage sélectionné. Les menaces seront séparées pour permettre la lecture de
symboles qui se chevauchent. Précision du roulement dégradée pour les menaces déplacées.

### Bouton de contrôle de l'intensité

Le bouton INT (<num>5</num>) ajuste la luminosité de l’écran.

## Avis de menace et voyants d'avertissement principaux

![maître des menaces](../../../img/cockpit_detail_rio_threatmaster.jpg)

Voyant principal d'avertissement et indications d'avertissement et d'avertissement ECM/IFF.

Le voyant MASTER CAUTION et le bouton de réinitialisation clignotent pour indiquer un changement
d'état sur
le panneau d’avertissement/consultatif RIO.

Appuyez pour accuser réception et éteindre la lumière jusqu'au prochain événement.

### Voyants d'avertissement ALR-67

| Indicateur | Fonction |
| --- | --- |
| IFF | Voyant indiquant une interrogation reçue en mode 4 sans que son propre système génère une réponse. |
| RCV | Voyant indiquant que l'ALQ-126 reçoit un signal d'identification de menace. |
| XMIT | Voyant indiquant que ALQ-126 est en train de transmettre. |
| SAM | Voyant d'avertissement, éclairage constant lors de la détection d'un verrouillage provenant d'un radar de suivi SAM. Clignote lorsqu'un lancement de missile est détecté. |
| AAA | Voyant d'avertissement, éclairage constant lors de la détection du verrouillage à partir d'un radar de suivi AAA. Clignote lorsque l’engagement AAA est détecté. |
| CW | Voyant lumineux indiquant la détection d'un émetteur d'ondes continues. |
| AI | Voyant d'avertissement, éclairage constant lors de la détection d'un verrouillage provenant d'un radar intercepteur aéroporté. |

### Voyants d'avertissement ALR-45

| Indicateur | Fonction |
| --- | --- |
| SA TRK | S'allume fixement chaque fois qu'un radar de suivi SAM est reçu. |
| SA2 | Avertissement SA-2 - S'allume fixe pour MA (alerte missile), clignote pour ML (lancement de missile). |
| SA3/NI | Avertissement SA-3 / SA-N-1 - S'allume fixe pour MA (alerte missile), clignote pour ML (lancement de missile). |
| SA4 | Avertissement SA-4 - S'allume fixe pour MA (alerte missile), clignote pour ML (lancement de missile). |
| AI/AAA | S'allume en continu chaque fois qu'un intercepteur aéroporté et/ou un radar AI/AAA ambigu est reçu. |
| REC | S'allume fixement chaque fois que l'ALQ-100 reçoit un signal identifié comme une menace. |
| IFF | Voyant indiquant une interrogation de mode 4 reçue sans que votre propre système génère une réponse. |
| SA6 | Avertissement SA-6 - S'allume fixe pour MA (alerte missile), clignote pour ML (lancement de missile). |
| AI | Les lumières restent fixes chaque fois qu'un radar intercepteur aéroporté est reçu. |
| REP | S'allume fixement chaque fois que l'ALQ-100 transmet. |
