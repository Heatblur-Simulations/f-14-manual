# Pré-démarrage

## Pilote

| Étape | Article | Action/Détails |
| --- | --- | --- |
| 1 | Ordonnez au commandant de bord de l'avion de connecter et d'appliquer l'air de démarrage et d'appliquer une alimentation externe | Attendez la confirmation. |
| 2 | Si les ailes ne sont pas en OV SWP : | |
| | | WING SWEEP DRIVE NO. 1 et WG SWP DRIVE NO 2/MANUV FLAP cb's (LE1, LE2) - Tirez. |
| | | Poignée d'urgence WINGSWEEP - Étendez et associez les barres de capitaine avec le ruban de positionnement des ailes. |
| | | **ATTENTION : Les ailes se déplaceront en position de poignée d'urgence quelle que soit la position du disjoncteur de balayage des ailes (cb).** |
| 3 | ICS | Vérifier |
| 4 | Indicateur de train d'atterrissage et feu de transition | Vérifier |
| | | Vérifiez l'indication de position de vitesse vers le bas et le voyant de transition éteint. |
| 5 | Commutateur MASTER TEST | Vérifier |
| | | - RPM - 96%. |
| | | - EGT - 960 °C. |
| | | - FF - 10500 Pph. |
| | | - AOA - 18±5. |
| | | - Balayage des ailes - 45°±2,5°. |
| | | - QTÉ DE CARBURANT - 2000 ± 200 livres. |
| | | - Quantité d'oxygène - 2 litres. |
| | | - Voyants L et R FUEL LOW - Allumés. |
| | | - Interrupteur MASTER TEST - OFF. |
| | | LTS - Vérifiez que tous les voyants d'avertissement, d'attention et de recommandation s'allument. Coordonner avec RIO. |
| | | Réglez la luminosité du panneau ACM et des lumières de l’indexeur pendant le test. |
| | | Le commutateur DATA LINK doit être allumé pour vérifier les voyants DDI. |
| | | FIRE DET/EXT - Vérifiez que les voyants L et R FIRE and GO s'allument. INST - Vérifiez les réponses suivantes sur les instruments après 5 secondes : |
| | | Fait clignoter les voyants L STALL et R STALL. |
| | | Vérifiez RIO QTÉ DE CARBURANT. |
| 6 | Sièges éjectables | Armé |
| | | Vérifiez que les sièges RIO sont armés. |
| 7 | Poignée CANOPY | Fermer |
| 8 | Panneau ACM | Ensemble |
| | | - Taux d'armes à feu - Réglez et vérifiez les cartouches restantes. |
| | | - SW COOL - OFF. |
| | | - MSL PREP - OFF. |
| | | - MSL MODE - NORM. |
| | | - Fenêtres d'état de chargement de la station - Vérifier. |
| | | Vérifier la cohérence avec les armements chargés. |
| 9 | EMERG STORES JETT bouton-poussoir lumineux | Éteint |
| 10 | Lumière ÉCHELLE | Éteint |
| | | Ordonnez au capitaine de l'avion de ranger l'échelle et les marches d'embarquement. |
| 11 | Informer RIO | Prêt à commencer |

## RIO

| Étape | Article | Action/Détails |
| --- | --- | --- |
| 1 | Vérifiez que l'alimentation externe et l'air sont connectés | **ATTENTION : Le démarrage du WCS et/ou des écrans sans air extérieur peut endommager ces systèmes, car l'air extérieur est utilisé pour le refroidissement lorsqu'aucun air à bord n'est disponible.** |
| 2 | ICS | Vérifier |
| | | Vérifiez la communication bidirectionnelle entre les membres de l'équipage et réglez le volume. |
| 3 | DL, TACAN et U/VHF | Réglé conformément aux paramètres de la mission. |
| 4 | Quantité de carburant | Vérifier |
| 5 | Lumières | Vérifier |
| | | Ajustez l’éclairage de la console et des instruments si nécessaire. |
| 6 | Test LTS | Vérifier |
| | | Vérifiez que tous les voyants d'avertissement et d'avertissement, les voyants ECM et les voyants DDI s'allument. |
| | | Ceci est effectué lorsque le pilote sélectionne le test LTS sur le commutateur MASTER TEST. |
| | | _Remarque :_ Lorsque le pilote sélectionne le test INST, le compteur de carburant RIO devrait diminuer à 2000 livres et les voyants MASTER CAUTION, FUEL LOW et OXY LOW devraient s'allumer. |
| 7 | Sièges éjectables | Armé |
| | | Vérifiez visuellement que la languette de verrouillage du pilote est enfoncée. |
| 8 | Poignée CANOPY | Fermer |
| | | Vérifiez que le pilote est prêt à fermer la verrière. |
| 9 | Reconnaître | Prêt à commencer |
| | | Informer le pilote, prêt à démarrer. |
