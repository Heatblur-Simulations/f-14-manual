# Démarrage du moteur

## Pilote

| Étape | Article | Action/Détails |
| --- | --- | --- |
| 1 | Vérifiez que le commutateur AIR SOURCE est réglé sur OFF | |
| 2 | Régler le commutateur HYD TRANSFER PUMP | OFF (Garde levée). |
| 3 | EMERG FLT HYD passer à LOW | Vérifiez le drapeau ON dans la fenêtre de pression hydraulique EMER FLT LOW. |
| | | Vérifiez le contrôle des gouvernes horizontales et du gouvernail sur l'indicateur de position de la surface. |
| 4 | EMERG FLT HYD passer à HIGH | Vérifiez l'indicateur ON dans la fenêtre de pression hydraulique de l'EMER FLT HI. |
| | Vérifiez le contrôle des gouvernes horizontales et du gouvernail sur l'indicateur de position de la surface. | Doit avoir un taux de déflexion supérieur à LOW. |
| 5 | EMERG FLT HYD commutateur sur AUTO (LOW) | Vérifiez les indicateurs OFF dans EMER FLT LOW et HI. |
| 6 | Commutateur ENG CRANK sur R (moteur droit). | Vérifiez la pression d’huile et hydraulique à environ 20 % RPM. |
| | | Vérifiez l’indication du voyant d’avertissement START/VALVE. |
| | | L'interrupteur est maintenu automatiquement en position par le solénoïde jusqu'à ce que le moteur atteigne environ 50 % du régime. |
| 7 | Accélérateur droit au RALENTI à 20 % RPM | Actionnera automatiquement le système d’allumage et le débit de carburant. |
| | | L'extinction de la lumière (augmentation de la température de l'EGT) devrait se produire dans les 5 à 15 secondes. |
| | | La température EGT doit culminer autour de 40 à 50 % et ne pas dépasser 890 °C, ce qui constitue un démarrage à chaud. |
| 8 | Vérifiez que l'interrupteur ENG CRANK revient sur OFF à environ 50 % du régime et que le voyant d'avertissement START/VALVE s'éteint | Si ENG CRANK reste sur R, désactivez-le manuellement avant 60 % tr/min. |
| | | Si le voyant d’avertissement START/VALVE est toujours allumé, débranchez l’air du démarreur. |
| 9 | Vérifiez que le voyant d'avertissement R GEN s'éteint | À environ 59% tr/min. |
| 10 | Vérifiez que le voyant d'avertissement R FUEL PRESS s'éteint | Avant le régime de ralenti. |
| 11 | Vérifier les lectures des instruments de ralenti du moteur | - RPM - 62 à 78%. |
| | | - EGT - 500 °C (nominal). |
| | | - FF - 950 à 1400 Pph (nominal). |
| | | - Position NOZ - 100%. |
| | | - HUILE - 25 à 35 psi (nominal, 15 minimum). |
| | | - FLT HYD PRESS - 3000 psi. |
| 12 | Ordonnez au commandant de bord de l'avion de débrancher l'alimentation électrique externe | |
| 13 | Commutateur ENG CRANK sur L (moteur gauche). | Lorsque la pression hydraulique combinée atteint 3 000 psi, remettez ENG CRANK sur OFF. |
| 14 | Commutateur HYD TRANSFER PUMP sur NORMAL. | Fonctionnera du côté vol pour maintenir le côté combiné entre 2 400 et 2 600 psi. |
| | S'il ne met pas sous pression le côté combiné dans les 10 secondes, réglez immédiatement le commutateur HYD TRANSFER PUMP sur SHUTOFF. | |
| 15 | Commutateur HYD TRANSFER PUMP sur SHUTOFF | |
| 16 | Répétez les étapes 6 à 11 pour le moteur gauche | |
| 17 | Ordonnez au commandant de bord de l'avion de débrancher l'air du démarreur | |
| 18 | Placez le commutateur AIR SOURCE sur L ENG, R ENG, puis réglez-le sur BOTH ENG | Vérifiez le flux d’air du cockpit à toutes les positions. |
| 19 | Commutateur HYD TRANSFER PUMP sur NORMAL | |

## RIO

Le RIO surveille les procédures du pilote et les signaux du capitaine de l'avion pour garantir un
maximum
sécurité pendant la séquence de démarrage du moteur.
