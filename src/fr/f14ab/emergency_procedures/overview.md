# Procédures d'urgence

## Démarrage pneumatique du moteur

Les moteurs F110-GE-400 peuvent être démarrés pneumatiquement un par un ou simultanément.
Trois méthodes de redémarrage sont disponibles :

- **Spool-down**
- **Saignement croisé**
- **Moulin à vent**

Toutes les tentatives doivent être effectuées dans le mode moteur sélectionné par l'AFTC
(normalement
PRI, ou SEC si PRI échoue).

### Démarrage pneumatique à bobine

Le démarrage pneumatique avec spoule-down doit être tenté immédiatement après la perte du moteur,
avant
un ralentissement important se produit.

- Peut prendre jusqu'à **10 secondes** pour augmenter le régime et **90 secondes** pour atteindre
régime commandé.
- Si PRI échoue, réessayez dans SEC.

#### Liste de contrôle pour le démarrage pneumatique spool-down

| Étape | Article | Actions |
| --- | --- | --- |
| 1 | Accélérateur | IDLE ou supérieur |
| 2 | Si aucun rallumage : Accélérateur | OFF puis IDLE |
| 3 | Commutateur ENG MODE SELECT | SEC |
| 4 | Accélérateur | Si aucun démarrage après le changement de mode : OFF puis IDLE |
| 5 | Après un démarrage réussi | Réglez ENG MODE SELECT sur PRI si possible |

### Démarrage pneumatique à purge croisée

À utiliser si :

- Échec du spool-down
- Un moteur tourne déjà
- Le moteur cible est proche du régime de l'éolienne.

#### Liste de contrôle de démarrage pneumatique à purge croisée

| Étape | Article | Actions |
| --- | --- | --- |
| 1 | Accélérateur qui ne fonctionne pas | OFF |
| 2 | Poignée FUEL SHUT OFF | Vérifier la poignée IN |
| 3 | Accélérateur du moteur en marche | Minimum 80 % tr/min |
| 4 | Commutateur BACK UP IGNITION | ON |
| 5 | Commutateur ENG CRANK | Régler sur moteur qui ne tourne pas |
| 6 | Accélérateur qui ne fonctionne pas | IDLE immédiatement après CRANK |
| 7 | Si pas de démarrage : Accélérateur | OFF puis IDLE |
| 8 | Commutateur ENG MODE SELECT | SEC |
| 9 | Accélérateur | OFF puis IDLE |
| 10 | Après un démarrage réussi | BACK UP IGNITION : OFF |
| 11 | Commutateur ENG MODE SELECT | PRI si possible |

### Démarrage pneumatique du moulin à vent

Utilisé lorsque :

- Vitesse ≥ 450 KIAS
- L'altitude permet une circulation d'air suffisante à travers le moteur

#### Liste de contrôle pour le démarrage pneumatique d'une éolienne

| Étape | Article | Actions |
| --- | --- | --- |
| 1 | Vitesse | À ou au-dessus de 450 KIAS |
| 2 | Accélérateur | IDLE ou supérieur |
| 3 | Commutateur BACK UP IGNITION | ON |
| 4 | Si aucun rallumage : Accélérateur | OFF puis IDLE |
| 5 | Commutateur ENG MODE SELECT | SEC |
| 6 | Accélérateur | OFF puis IDLE |
| 7 | Après un démarrage réussi | BACK UP IGNITION : OFF |
| 8 | Commutateur ENG MODE SELECT | PRI si possible |
