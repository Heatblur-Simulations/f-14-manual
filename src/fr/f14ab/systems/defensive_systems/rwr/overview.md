# Récepteur d'alerte radar

Deux ensembles de récepteurs d'avertissement radar sont disponibles pour le Tomcat. L'ALR-45/50
et le dernier ALR-67.

Ils sont capables de détecter passivement les menaces radar, augmentant ainsi la
conscience situationnelle de l'équipage.
