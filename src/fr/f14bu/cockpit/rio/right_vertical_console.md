# Console verticale droite

## Affichage des contre-mesures électroniques (ECMD)

![ECMD](../../../img/bu/right_vertical_console_electronic_countermeasure_display.jpg)

L'ECMD (<num>1</num>), un bouton de contrôle de la luminosité (<num>2</num>), un bouton de test et
un indicateur BIT indiquant l'état de l'afficheur (noir fixe lorsqu'il est opérationnel,
affichant des drapeaux blancs lors de l'indication d'une condition d'échec).

L'ECMD est contrôlé via le panneau de commande ECMD (<num>3</num>). Pour une description
détaillée de l'ECMD, reportez-vous à la
[PMDIG Section](../../systems/pmdig/overview.md).
