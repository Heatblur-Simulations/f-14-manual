## Plancher gauche

### Chargeur de données de mission AN/ASQ-215

Le [chargeur de données de mission](../../systems/mdl/overview.md) (<num>1</num>) (MDL)
contient le module de transfert de données (DTM). Le DTM stocke toutes les données spécifiques à la
mission, telles que les waypoints, les coordonnées cibles GGW et les profils CMS. Le MDC peut être
éjecté en appuyant sur le bouton situé sous la cartouche (<num>2</num>).

### Bouton du pied gauche ICS

RIO repose-pied gauche (<num>3</num>) contenant ICS PTT pour COLD MIC
intercommunication.

### Panneau consultatif de mise en garde

Les voyants d'avertissement (<num>1</num>) s'allument pour avertir des conditions dangereuses qui
nécessitent des mesures correctives immédiates. Les voyants d'avertissement affichent des lettres
jaunes sur un
fond opaque pour indiquer une condition dangereuse imminente. La moitié inférieure
du panneau ATTENTION ADVISORY se compose de voyants d'avertissement qui s'allument en vert
lettres sur fond opaque. Les voyants lumineux indiquent des opérations dégradées qui
peut nécessiter des mesures correctives.

| Indicateur | Fonction |
| --- | --- |
| C&D CHAUD | Voyant d'avertissement indiquant une surchauffe dans les commandes et les affichages RIO. |
| CABIN PRESS | Voyant d'avertissement indiquant que la pression dans la cabine est trop basse. |
| FUEL LOW | Témoin d'avertissement indiquant un niveau de carburant inférieur à 1 000 livres dans les groupes d'alimentation en carburant arrière et gauche ou avant et droit. |
| OXY LOW | Voyant d'avertissement indiquant que la quantité d'oxygène est inférieure à 2 litres ou que la pression est trop faible. |
| CANOPY | Voyant d'avertissement indiquant que l'auvent n'est pas abaissé et verrouillé. |
| FUSE HV | Voyant d'avertissement indiquant que le fusible électrique AWW-4 ne fonctionne pas. |
| RDR ENABLED | Voyant d'avertissement indiquant que le fonctionnement du radar avec poids sur roues est possible. |
| COOLING AIR | Voyant indiquant une surchauffe dans le système électronique de refroidissement à air pulsé. |
| MSL COND | Voyant indiquant une surchauffe ou une sous-pression dans le flux de liquide de refroidissement du missile, l'une ou l'autre arrête la pompe de liquide de refroidissement du missile. Peut également indiquer que l'interrupteur LIQ COOLING n'est pas en position AWG-9/AIM-54 avec le WCS en STBY ou ON lorsque les carénages Phoenix sont installés. |
| AWG-9 COND | Voyant indiquant une surchauffe ou une surpression dans le débit de liquide de refroidissement AWG-9 ou que le commutateur de surchauffe a arrêté la pompe de liquide de refroidissement. |
| NAV COMP | Voyant indiquant une panne dans le INS ou le CSDC avec le commutateur NAV MODE dans INS. |
| FILM LOW | Voyant lumineux indiquant une faible quantité restante de film de l'enregistreur de mission. |
| IMU | Voyant indiquant une panne de la centrale inertielle ou que le système de navigation est en mode AHRS/AM. |
| AHRS | Feu consultatif indiquant que les informations d'assiette ou de cap du AHRS ne sont pas fiables. |

### Bouton du pied droit ICS

RIO repose pied droit (<num>2</num>) contenant le PTT pour transmission sur UHF 1,
V/UHF 2, ou les deux selon le paramètre ICS.
