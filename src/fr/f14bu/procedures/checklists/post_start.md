# Post-démarrage

## Post-démarrage (pilote)

| Étape | Article | Action/Détails |
| --- | --- | --- |
| 1 | Commutateurs STAB AUG | Tout est allumé |
| 2 | Commutateur MASTER TEST | Réglé sur EMERG GEN |
| 3 | Bouton-poussoir MASTER RESET | Appuyer sur <br> Vérifier que les voyants d'avertissement DFCS sont éteints |
| 4 | Commutateur MASTER TEST | OFF |
| 5 | Bouton-poussoir MASTER RESET | Appuyer sur <br> Vérifier que les voyants d'avertissement DFCS sont éteints. Les commutateurs STAB AUG ne doivent pas se désengager. |
| 6 | Notifier RIO | Informer que le test est terminé |
| 7 | Commutateurs STAB AUG | Tout est allumé |
| 8 | AFTC | L/R ENG MODE SELECT : SEC → PRI |
| 9 | Poignée d'urgence WING SWEEP | S'il n'est pas en surbalayage :<br>Déplacez-vous à 68°<br>Puis OV SW complet<br>Maintenez jusqu'à ce que le voyant HZ TAIL AUTH s'éteigne et que l'indicateur OVER apparaisse<br>Puis rangez |
| 10 | Commutateur de mode de balayage des ailes | AUTO |
| 11 | Disjoncteurs WING SWEEP DRIVE NO.1 et WG SW DR NO.2/MANUV FLAP | Enclenchés |
| 12 | Commutateur WING/EXT TRANS | OFF |
| 13 | Bouton-poussoir MASTER RESET | Appuyer sur |
| 14 | COMM/NAV/GEAR/DISPLAYS | Réglez tout sur ON :<br>- UHF sur TR+G ou LES DEUX<br>- TACAN sur T/R<br>- ARA-63 sur ON<br>- DISPLAYS les interrupteurs d'alimentation ON<br>- RADAR ALT ON |
| 15 | Trim | Réglé sur 000 |
| 16 | Gyroscope d'attitude de veille | Érigé |
| 17 | Bouton-poussoir MASTER RESET | Appuyer sur |
| 18 | Commutateur MASTER TEST | Définir sur OBC |
| 19 | Pilote automatique | S'engager |
| 20 | Commutateur de frein de vitesse | Étendre → Rétracter |
| 21 | Commutateur REFUEL PROBE | Étendre → Rétracter |
| 22 | Commutateur WSHLD AIR | Cycler |
| 23 | OBC | OFF |
| 24 | Commutateur WING/EXT TRANS | OFF |
| 25 | Trim | Revérifiez et réglez sur 000 |
