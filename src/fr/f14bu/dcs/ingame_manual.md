# Manuel intégré

Ce manuel est intégré au jeu et peut être lu en vol.

Il offre également un moyen d'expliquer directement les commutateurs du cockpit en ouvrant le
manuel déjà positionné sur l'endroit expliquant le commutateur correspondant.

## Contrôles

La liaison par défaut <kbd>M</kbd> peut être utilisée pour basculer le manuel. En la maintenant
enfoncée tout en cliquant sur un interrupteur dans le cockpit, le manuel s'ouvrira automatiquement
et défilera jusqu'à la section expliquant ce commutateur.

De plus, les liaisons suivantes sont également disponibles :

- <kbd>RSHIFT</kbd> + <kbd>M</kbd> - ouvrir le manuel
- <kbd>RCTRL</kbd> + <kbd>M</kbd> - fermer le manuel
- <kbd>RALT</kbd> + <kbd>M</kbd> - basculer vers le manuel

![Exemple d’outil de liste de contrôle](../../img/bu/ingame_manual_example.jpg)

> 💡 Afin de fermer le manuel, assurez-vous d'abord de retirer le focus clavier du manuel
> en cliquant n'importe où ailleurs dans le cockpit.
