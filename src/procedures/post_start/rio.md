# 起动后 (RIO)

> 💡 注意：RIO 必须在开始“起动后”检查单之前完成 EMERG GEN 检查
> 检查单。在航电设备和 AWG-9 通电之前，ECS（环境控制系统）应该运行至少3分钟
> 以及 AWG-9 通电。

| 步骤 | 项目                              | 动作/细节                                                             |
| ---- | --------------------------------- | --------------------------------------------------------------------- |
| 1    | WCS 选择开关                      | STBY                                                                  |
|      |                                   | 核实 AWG-9 COND 灯亮起。                                              |
| 2    | LIQ COOLING 开关                  | AWG-9 或 AWG-9/AIM-54                                                 |
|      |                                   | 如果挂载 AIM-54 则选择 AWG-9/AIM-54。                                 |
|      |                                   | 核实 AWG-9 COND 灯熄灭，且 AUTO BIT 2 正在运行（在 TID 上）。         |
| 3    | IR/TV 电源开关                    | STBY/IR/TV                                                            |
| 4    | CATEGORY 选择旋钮                 | NAV                                                                   |
| 5    | NAV MODE 开关                     | 对准                                                                  |
| 6    | 通信                              | ON 并调定                                                             |
|      |                                   | - U/VHF 模式选择旋钮 - T/R G。                                        |
|      |                                   | - DATA LINK 开关 - ON。                                               |
| 9    | TACAN 模式选择旋钮                | T/R                                                                   |
| 10   | RADAR WARNING RCVR 面板           | 调定                                                                  |
|      |                                   | - DISPLAY TYPE 选择开关 - NORM。                                      |
|      |                                   | - PWR 开关 - ON。                                                     |
|      |                                   | - TEST 开关 - SPL。                                                   |
|      |                                   | - MODE 按钮 - LMT。                                                   |
| 11   | DECM 旋钮                         | STBY                                                                  |
|      |                                   | - 当 STBY 指示灯熄灭时，选择 HOLD 3 SEC 档位，然后选择 ACT 进行 OBC。 |
| 12   | IFF MASTER 旋钮                   | STBY                                                                  |
|      |                                   | - 调定 CODE 旋钮 - 按需。                                             |
|      |                                   | - IFF 控制面板 - 测试。                                               |
|      |                                   | - MC 开关 - OUT。                                                     |
|      |                                   | - M1、M2 和 M3 - 测试。                                               |
|      |                                   | - MC - 测试。                                                         |
|      |                                   | - IFF ANT 开关 - AUTO。                                               |
|      |                                   | 选择 NORM 并观察 TEST 指示灯                                          |
|      |                                   | 亮起。                                                                |
| 13   | AUTO BIT 2                        | 核实完成                                                              |
|      |                                   | 在 TID 上。                                                           |
| 14   | CAP                               | 输入本机所处的纬度，经度以及机场或母舰高度                            |
| 15   | 高度表                            | 复位                                                                  |
| 16   | 计算机位址面板                    | 输入所需数据                                                          |
|      |                                   | 例如 WP FP 等，根据任务需求 。                                        |
| 17   | DDD                               | 调定                                                                  |
| 18   | TID 控制开关/按钮                 | 调定                                                                  |
|      |                                   | - CONTRAST 旋钮 - 调定。                                              |
|      |                                   | - BRIGHT 旋钮 - 调定。                                                |
|      |                                   | - CLSN 按钮 - OFF。                                                   |
|      |                                   | - ALT NUM 按钮 - ON。                                                 |
|      |                                   | - SYM ELEM 按钮 - ON。                                                |
|      |                                   | - DATA LINK 按钮 - 按需。                                             |
|      |                                   | - JAM STROBE 按钮 - 按需。                                            |
|      |                                   | - NON ATTK 按钮 - 按需。                                              |
|      |                                   | - LAUNCH ZONE 按钮 - 按需。                                           |
|      |                                   | - VEL VECTOR 按钮 - 按需。                                            |
|      |                                   | - RANGE 选择开关 - 按需。                                             |
| 19   | 多功能显示指示器                  | 调定                                                                  |
|      |                                   | - TEST 按钮 - 按下并检查。                                            |
|      |                                   | - 亮度 - 调定。                                                       |
| 20   | DATA/ADF 开关                     | BOTH                                                                  |
| 21   | HCU 控制面板                      | 调定                                                                  |
|      |                                   | - 灯光测试 - 按下并检查。                                             |
|      |                                   | - ELEV 游标 - 在雷达侧调定为0°。                                     |
|      |                                   | 所有 AWG-9 指示灯应亮起。                                             |

|      |                                   | - BURST 开关 - 3。                                                    |

| 24   | D/L 应答                          | 按需                                                                  |

|      |                                   | - TEST/CHAL CC 开关 - 测试。                                          |
|      |                                   | 检查 DDD 显示。                                                       |

|      |                                   | - PRGM RESTRT 按钮 - 按下。                                           |
|      |                                   | - STBY/READY 指示灯 - OFF。                                           |
|      |                                   | - TID NAV 模式 - INS。                                                |
| 29   | DEST 数据                         | 核实                                                                  |
| 30   | 到目标点的方位/距离               | 检查                                                                  |
| 31   | 本机地速                          | 检查                                                                  |
| 32   | 磁差                              | 检查                                                                  |
