# 右膝仪表板

## 注意-提示面板

![caution advisory](../../img/cockpit_detail_rio_cautionadvisory.jpg)


| 指示器      | 功能                                                                                                                                                                                                                                            |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| C&D HOT     | “控制和显示器过热”注意灯，灯光亮起表示 RIO 控制和显示器过热。                                                                                                                                                                                 |
| CABIN PRESS | “座舱压力”注意灯，灯光亮起表示驾驶舱压力过低。                                                                                                                                                                                                |
| FUEL LOW    | “油量低”注意灯，RIO 驾驶舱中的 FUEL LOW 将随飞行员驾驶舱中任意一侧 FUEL LOW 注意灯同时亮起。                                                                                                                                                  |
| OXY LOW     | “供氧压力低”注意灯，灯光亮起表示剩余液氧储量少于 2 公升，或压力过低。                                                                                                                                                                         |
| CANOPY      | “座舱盖”注意灯，灯光亮起表示座舱盖未降下并锁定。                                                                                                                                                                                              |
| FUSE HV     | “引信过电压”注意灯，灯光亮起表示 AWW-4 电子引信不工作。                                                                                                                                                                                       |
| RDR ENABLED | “雷达启用”注意灯，灯光亮起表示雷达可在机轮负重情况下进行操作。                                                                                                                                                                                |
| COOLING AIR | “冷气通风”提示灯，灯光亮起表示电力强制空气冷却系统超温状态。                                                                                                                                                                                  |
| MSL COND    | “导弹状态”提示灯，灯光亮起表示导弹冷却液流超温或欠压，且两种情况都会导致导弹的冷却液泵关闭。如果“不死鸟”导弹挂架安装了整流罩，状态提示灯光亮起也表示 WCS 正处于 STBY（待机）或 ON（开机）状态，但 LIQ COOLING 开关未拨至 AWG-9/AM-54 档位。 |
| AWG-9 COND  | “AWG-9 状况”提示灯，灯光亮起表示 AWG-9 冷却液流过热或压力过低，或当超温开关关闭冷却泵时亮起。                                                                                                                                                 |
| NAV COMP    | “导航组件”提示灯，NAV MODE 选择开关位于 INS 档位时，灯光亮起表示 INS 或 CSDC 故障。INS 正在降级模式下运行。                                                                                                                                   |
| FILM LOW    | “胶卷余量”提示灯，灯光亮起表示机载任务录像机胶片余量低。                                                                                                                                                                                      |
| IMU         | “惯性测量装置”提示灯，灯光亮起表示惯性测量装置故障，或导航系统处于 AHRS（姿态航向基准系统）/AM（大气数据）模式。                                                                                                                              |
| AHRS        | “姿态航向基准”提示灯，灯光亮起表示 AHRS 提供的姿态或航向信息不可靠。                                                                                                                                                                          |
