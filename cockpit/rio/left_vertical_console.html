Heatblur F-14 Tomcat

Left Vertical Console

Armament Panel

The armament panel provides primary control of air-to-air and air-to-ground weapon employment from the RIO cockpit.

Weapon Type Selector

The WPN TYPE selector (1) selects the weapon type used by the WCS for air-to-ground ballistic calculations.

Attack Mode Selector

The ATTK MODE knob (2) selects the air-to-ground attack mode used for weapon delivery.

Electrical Fuse Selector

The ELEC FUSE knob (3) selects the electrical fuze setting for air-to-ground ordnance.

Air-to-Ground Gun Switch

The A/G GUN switch (4) selects gun availability in A/G master mode.

  • OFF - Gun disabled.
  • MIXED - Enables gun in addition to selected A/G ordnance.

Quantity Selectors

The QTY selector wheels (5) set the quantity of air-to-ground ordnance released per delivery, including rockets.

Interval Selectors

The INTERVAL selector wheels (6) set the time interval between successive weapon releases in ripple delivery mode.

Interval is set in milliseconds.

Station 6 Select Switch

The station 6 select switch (7) selects station 6 for jettison or air-to-ground weapons delivery.

Air-to-Air Launch Button

The A/A LAUNCH button (8) allows the RIO to command launch of AIM-7 or AIM-54 missiles.

Button illumination indicates hot-trigger conditions are met.

Missile Speed Gate Knob

The MSL SPD GATE knob (9) controls the position of the missile speed gate used by the WCS.

Station 8 Select Switch

The station 8 select switch (10) selects station 8 for jettison or air-to-ground weapons delivery.

  • B - Selects lower pylon.
  • SW - Non-functional.

Missile Options Switch

The MSL OPTIONS switch (11) configures missile launch behavior.

  • Enables AIM-7 pulse-doppler mode.
  • Enables AIM-54 active launch mode.

Station 5 Select Switch

The station 5 select switch (12) selects station 5 for jettison or air-to-ground weapons delivery.

Next Launch Button

The NEXT LAUNCH button (13) designates the currently hooked target as the next launch target during TWS operation.

Station 4 Select Switch

The station 4 select switch (14) selects station 4 for jettison or air-to-ground weapons delivery.

Station 1 Select Switch

The station 1 select switch (15) selects station 1 for jettison or air-to-ground weapons delivery.

  • B - Selects lower pylon.
  • SW - Non-functional.

Tank Jettison Station 7 Switch

The TANK JETT station 7 switch (16) selects station 7 external fuel tank for jettison.

Tank Jettison Station 2 Switch

The TANK JETT station 2 switch (17) selects station 2 external fuel tank for jettison.

Station 3 Select Switch

The station 3 select switch (18) selects station 3 for jettison or air-to-ground weapons delivery.

Jettison Options Switch

The JETT OPTIONS switch (19) selects jettison configuration.

  • WPNS - Jettison weapons only.
  • MER/TER - Jettison racks in addition to weapons.

This function is non-functional in the modeled F-14.

Selective Jettison Switch

The SEL JETT switch (20) selects selective jettison mode.

  • JETT - Normal jettison mode.
  • AUX - Backup jettison mode (guarded).

Mechanical Fuse Selector

The MECH FUSE switch (21) enables and selects the mechanical fuze setting for air-to-ground ordnance.

Delivery Mode Selectors

The DLVY MODE switches (22) consist of two selector switches that control air-to-ground release logic.

  • One switch selects single or paired weapon release.
  • The other selects single-pass or multiple-pass delivery according to quantity and interval settings.
